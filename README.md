# Lagrange.Installer
> #### 一个针对 [Lagrange.OneBot](https://github.com/LagrangeDev/Lagrange.Core) 的安装脚本

## 💻如何安装

1. 下载最新的 [releases](https://github.com/xiaosuyyds/Lagrange.Installer/releases)
2. 运行 `Installer`
3. 等待安装完成
